FlickrDown Help

Getting Started

Step 1: Type in a flickr username, e-mail address, group or tags and click "Lookup..."

Step 2: If you want to download an entire set then click the checkbox beside that set. If you want to download just a few selected photos click the name of the set and all the photos for that set will appear. Check the images you want to download

Step 3: Click the "Browse..." button and select a folder to download to.

Step 4: Click "Download"

Authorization

Authorization lets flickrdown see private pictures.

Notes

  • Photos will be download to a subfolder named after the set they are in
  • The [*All_Photos*] set is for photos not in a photoset.
  • Setting the Download Limit to a high number could make Flickrdown take a REALLY long time. Please do not abuse this setting. For example setting it 1000000 and downloading all images with the "flowers" tag will probably get you banned from flickr
